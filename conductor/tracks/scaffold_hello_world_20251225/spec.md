# Spec: Scaffold a ''Hello World'' full-stack application

This track will set up a minimal, end-to-end application with a database, a FastAPI backend, and a React frontend.

## Key Deliverables to it locally
- A PostgreSQL database with a 'messages' table.
- A FastAPI backend serving an API to retrieve messages.
- A React frontend to display the messages.

## Key Deliverables to it remotely on dev
- A PostgreSQL database with a 'messages' table.
- A FastAPI backend serving an API to retrieve messages.
- A React frontend to display the messages.
